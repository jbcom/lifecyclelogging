================================
Contributing Guide
================================

Development Setup
---------------

1. Clone the repository
2. Create development environment:

   .. code-block:: bash

      make env

3. Run tests:

   .. code-block:: bash

      make test

4. Format code:

   .. code-block:: bash

      make format

Pull Request Guidelines
--------------------

1. Add tests for new features
2. Update documentation
3. Update CHANGELOG.md
4. Ensure all checks pass:

   .. code-block:: bash

      make check
